# HTML & CSS — Projetos

Repositório com desafios e mini-projetos desenvolvidos durante meus estudos de HTML5 e CSS3, com base no curso do Curso em Vídeo (Gustavo Guanabara).

## Objetivo

Consolidar o aprendizado em HTML e CSS por meio de desafios práticos e servir como registro da minha evolução no desenvolvimento front-end.

## Conteúdos praticados

- Fundamentos de desenvolvimento web  
- Estruturação semântica com HTML5  
- Estilização e layout com CSS3  
- Tipografia, cores e identidade visual  
- Layout responsivo e adaptação a diferentes telas  
- Organização de projetos e boas práticas de código  
- Versionamento de código com Git e GitHub
